<Tip>
  Before running any code, confirm that a CUDA-capable GPU is available:

  ```python
  import torch
  print(torch.cuda.is_available())  # must return True
  ```
</Tip>

<Steps>
  <Step title="Install FlashAttention">
    ```bash
    pip install flash-attn --no-build-isolation
    ```

    See [Installation](/installation) for full requirements and options including ROCm, FA3, and FA4.
  </Step>

  <Step title="Basic forward pass">
    <Tabs>
      <Tab title="FA2">
        ```python
        import torch
        from flash_attn import flash_attn_func

        batch, seqlen, nheads, headdim = 2, 1024, 16, 64

        q = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")
        k = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")
        v = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")

        out = flash_attn_func(q, k, v)
        print(out.shape)  # torch.Size([2, 1024, 16, 64])
        ```
      </Tab>
      <Tab title="FA4">
        ```python
        import torch
        from flash_attn.cute import flash_attn_func

        batch, seqlen, nheads, headdim = 2, 1024, 16, 64

        q = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")
        k = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")
        v = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")

        out = flash_attn_func(q, k, v)
        print(out.shape)  # torch.Size([2, 1024, 16, 64])
        ```
      </Tab>
    </Tabs>

    <Note>
      FA2 only accepts `torch.float16` (fp16) or `torch.bfloat16` (bf16) inputs. `torch.float32` is not supported and will raise an error. bf16 requires an Ampere, Ada, or Hopper GPU.
    </Note>
  </Step>

  <Step title="Causal attention">
    Pass `causal=True` to apply a causal (autoregressive) mask, where each token only attends to itself and earlier positions. This is the standard setting for decoder-only language models.

    <Tabs>
      <Tab title="FA2">
        ```python
        import torch
        from flash_attn import flash_attn_func

        batch, seqlen, nheads, headdim = 2, 1024, 16, 64

        q = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")
        k = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")
        v = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")

        out = flash_attn_func(q, k, v, causal=True)
        ```
      </Tab>
      <Tab title="FA4">
        ```python
        import torch
        from flash_attn.cute import flash_attn_func

        batch, seqlen, nheads, headdim = 2, 1024, 16, 64

        q = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")
        k = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")
        v = torch.randn(batch, seqlen, nheads, headdim, dtype=torch.float16, device="cuda")

        out = flash_attn_func(q, k, v, causal=True)
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Check the output shape">
    The output tensor has the same shape as `q`: `(batch, seqlen, nheads, headdim)`.

    ```python
    assert out.shape == (batch, seqlen, nheads, headdim)
    # torch.Size([2, 1024, 16, 64])
    ```

    The last dimension of all input tensors must be contiguous in memory and 16-byte aligned.
  </Step>
</Steps>

## FA2 vs FA4 at a glance

| | FA2 | FA4 |
|---|---|---|
| Import | `from flash_attn import flash_attn_func` | `from flash_attn.cute import flash_attn_func` |
| Package | `flash-attn` | `flash-attn-4` |
| GPU targets | Ampere, Ada, Hopper, ROCm | Hopper (SM90), Blackwell (SM100/SM110) |
| Kernel | Precompiled CUDA C++ | JIT-compiled CuTeDSL (PTX/CUBIN) |
| Status | Stable | Active development |
